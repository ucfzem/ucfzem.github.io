## Features

- ✏️ **Inline editor** — edit brand name, hero text, gallery, features, colors
- 🌍 **4 languages** — English, French, Spanish, Arabic (RTL)
- 🖼️ **24 photo slots** with batch upload & lightbox
- 🔐 **Password lock** — prevent editing when page is live
- 🎨 **Live color picker** — 6 CSS tokens, instant preview
- ⬇️ **Download as HTML** — save your work as a standalone file
- 🔗 **Share link** — copy page URL to clipboard
- 💾 **localStorage persistence** — edits survive refresh
- 📱 **Responsive** — 4-column grid → 3 → 2 on mobile
- 🌀 **Scroll reveal animations**

## How to use

1. Open the **[live demo](https://ucfzem.github.io/Brand-Landig-Page/)**
2. Click the **✏️ Edit** button (bottom-right)
3. Customize text, colors, photos, logo
4. Lock the page with a password when done
5. Download the HTML or share the link

## Tech

Single `index.html` — vanilla HTML/CSS/JS. No build step, no dependencies. Works on GitHub Pages.
