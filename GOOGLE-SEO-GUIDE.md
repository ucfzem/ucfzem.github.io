# Google Search Console — Guide

## Valider les projets restants

Pour chaque URL :
1. https://search.google.com/search-console
2. Menu haut gauche → Ajouter une propriété
3. Coller l'URL → Préfixe d'URL → Continuer
4. Balise HTML (déjà dans le code) → VALIDER

https://ucfzem.github.io/contractforge/
