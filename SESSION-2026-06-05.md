## NanoLink 🔗

URL shortener with QR codes + analytics, déployé sur Cloudflare Pages + D1.

### Problèmes corrigés

1. **Bug demo mode** — `isDemoMode` détectait `localhost`/`127.0.0.1` uniquement → plantait sur `nanolink-6nx.pages.dev`. Fixé à `false` (mode LIVE permanent).

2. **Bug `_worker.js` fantôme** — Un `_worker.js` (33KB) avec HTML embarqué traînait dans le dossier et override toutes les Pages Functions. Supprimé.

3. **D1 Database** — Créée via Cloudflare Dashboard. ID : `26026c32-50fc-4be4-bf4e-230b4ca88787`. Schéma appliqué (tables `links` + `click_events`).

4. **Wrangler config** — `account_id` supprimé du `wrangler.toml` (non supporté par Pages). `compatibility_date` mis à `2026-06-05`. Projet créé proprement sous `nanolink-v2`.

5. **Pages Functions** — 4 functions déployées avec wrangler (pas direct upload) :
   - `functions/api/shorten.js` — POST: crée un lien
   - `functions/api/links.js` — GET: liste les liens + DELETE: efface tout
   - `functions/api/stats.js` — GET: statistiques globales + clics par jour
   - `functions/[slug].js` — GET: redirige 301 vers l'URL cible

## MicroInvoice 🧾

Générateur de factures PDF en ligne, multilingue, sans backend.

### Stack

- HTML + CSS + JS vanilla
- jsPDF + jspdf-autotable pour l'export PDF
- Zéro backend, zéro dépendance serveur
- Déployé sur Cloudflare Workers (static assets)

### Fonctionnalités

- **4 langues** : AR · FR · EN · ES
- **TVA Maroc** : 20%, 14%, 10%, 7%, 0% (Exonéré)
- **Devises** : MAD (درهم) / EUR (€)
- **Aperçu en direct** — les totaux se mettent à jour instantanément
- **PDF** avec tableau, TVA, totaux
- **Bouton "🎲 Démo"** — préremplit avec un exemple fictif

### Design

- Thème **marron-doré** (#d4a373)
- Fond brun foncé (#0d0a08)
- Interface responsive, fonctionne sur mobile
