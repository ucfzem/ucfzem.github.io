## 1. Balises SEO ajoutées au blog (`blog/index.html`)

### Meta tags de base
- `<meta name="description" content="...">` — description en français
- `<meta name="robots" content="index, follow">` — autorise l'indexation

### Open Graph (Facebook, LinkedIn, WhatsApp)
- `og:title`, `og:description`, `og:url`, `og:type: website`, `og:image`

### Twitter Cards
- `twitter:card: summary_large_image`, `twitter:title`, `twitter:description`

### Techniques
- `<link rel="canonical" href="https://ucfzem.github.io/blog/">`
- `<link rel="alternate" hreflang="fr/en/es/ar" href="...?lang=...">` — 4 liens + x-default
- `<meta name="google-site-verification" content="WF34yBc_q67rq4JfrEp-ZOT0pvqQWLf7uY6WbHu7Uig">` (Cloudflare Workers)

### JSON-LD Structured Data
```json
{
  "@context": "https://schema.org",
  "@type": "Blog",
  "name": "UcfZemTech",
  "description": "Blog tech couvrant l'IA, l'e-commerce et le SaaS",
  "url": "https://ucfzem.github.io/blog/",
  "inLanguage": ["fr", "en", "es", "ar"],
  "author": { "@type": "Person", "name": "Uzem", "url": "https://ucfzem.github.io/" },
  "image": "https://ucfzem.github.io/images/avatar.png"
}
```

### Langue URL sync
Le JS de changement de langue met désormais à jour l'URL (`?lang=en` etc.) via `history.replaceState`, et détecte d'abord le paramètre URL avant `localStorage`.

**Fichier :** `blog/index.html` (lignes 1-24, 1125-1145)
