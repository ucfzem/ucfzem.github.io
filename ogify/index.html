📄 API

Endpoint

GET /og?title=...&key=...

Paramètres

titleRequis. Texte principal (max 200 car.)
subtitleOptionnel. Texte secondaire
bgOptionnel. URL image de fond
logoOptionnel. URL logo (72×72, arrondi)
keyRequis. Clé API

Exemple cURL

curl -o og-image.png "https://ogify-api.azer-tyu199p.workers.dev/og?title=Ogify+votre+API+OG&subtitle=Généré+en+temps+réel&key=demo"

Code response

200 → image/png (cache 1 an)
400 → { "error": "title is required" }
429 → { "error": "rate_limit" }
500 → { "error": "generation_failed" }
